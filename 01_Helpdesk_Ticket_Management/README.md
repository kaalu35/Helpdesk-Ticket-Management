# Helpdesk Ticket Management - Prebuilt Demo App

## Purpose
This demo application is for practicing real-time business application understanding, requirement analysis, design review, and QA testing.

## How to Run
Open:

```text
src/index.html
```

No installation is required. It is a static HTML, CSS, and JavaScript demo.
